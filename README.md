# arturfilipe.pt — Portfolio de Cibersegurança

Site pessoal de **Artur Filipe Custódia**, profissional em transição para cibersegurança.

## ✨ Sobre

Este site serve como portfólio e CV online, listando projetos práticos de cibersegurança baseados na lista do canal **Mad Hat**, organizados do mais fácil/essencial ao mais avançado.

## 🛠 Tecnologias

- HTML5 + CSS3 (design limpo e responsivo)
- Inter (Google Fonts)
- GitHub Pages (hospedagem)

## 🚀 Deploy

O site é servido via **GitHub Pages** a partir do branch `main`.

1. Cria um repositório no GitHub com o nome `arturfilipe.pt`
2. Segue as instruções para fazer push:
```bash
git remote add origin git@github.com:ArturFilipe/arturfilipe.pt.git
git branch -M main
git push -u origin main
```
3. No GitHub, vai a Settings > Pages e seleciona o branch `main` como source
4. (Opcional) Configura o domínio personalizado `arturfilipe.pt` nas definições do Pages
